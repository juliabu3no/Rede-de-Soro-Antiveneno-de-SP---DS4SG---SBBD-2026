# Caracterização da Rede de Soro Antiveneno de São Paulo: Uma Análise a Partir de Dados Públicos

Projeto desenvolvido no âmbito do **Data Science for Social Good Brazilian Workshop (DS4SG)**, realizado durante o **41º Simpósio Brasileiro de Banco de Dados (SBBD)**.

## Sobre o projeto

Este repositório reúne os notebooks e códigos utilizados na análise dos acidentes com animais peçonhentos no estado de São Paulo, utilizando dados do SINAN/DATASUS (2007–2025).

O estudo busca compreender a demanda por atendimento soroterápico, a distribuição espacial dos casos e os fluxos de pacientes entre municípios.

## Objetivos

* Analisar a frequência de acidentes por tipo de animal;
* Avaliar a gravidade dos casos e o uso de soroterapia;
* Investigar o tempo entre acidente e atendimento;
* Mapear a distribuição geográfica dos acidentes;
* Identificar deslocamentos intermunicipais e polos de atendimento.

## Dados

As análises utilizam principalmente:

* Dados de acidentes com animais peçonhentos do SINAN/DATASUS;
* Informações sobre municípios e regionalização da saúde;
* Dados dos Pontos Estratégicos para Soro Antiveneno (PESA);
* Bases cartográficas e dados populacionais.
